Permissions
How to make a script executable: 
                     chmod file name.sh
                     chmod +x filename.sh
to be able to write inside a file: nano filename.sh
to be able to read what wass inside your file: sh file name.sh
to see the permissions a file has:ls -la file name.sh

permission to read:chmod 400 filename.sh
permission to write:chmod 200 filename.sh
permission to execute:chmod 100 filename.sh
all permission: chmod 777 filename.sh

print permissions:echo "file permissions are:">> ls -l
