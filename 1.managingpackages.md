Linux commands and concepts.

The packaging manager in managing packages in ubuntu is: sudo apt update
command to change directory/ get to desktop: cd desktop

create a folder:mkdir name

to enter a folder:cd folder name

pwd:print working directory

to come back behind: cd .. or cd ../..

rename folder:mv practice folder name

create file: touch "file name"

rename a file:mv index file name

to know the current user:whoami

absolute path:c:/users/hp/image/image name

to move files/folders: mv /c/users/hp/desktop/folder name/file name  /cmt/c/users/hp/desktop/folder name/file name  (thesame procedure applies for folders)

how to remove files:rm file name

remove everything from the folder: rm *.html/any other extensions(in case there are only html files)

delete all types of files:rm file*

remove folders:rm -r folder name

view an edit file(install nano):sudo apt install nano

read/see the content of a file: cat file name.extension

write inside a file directly: echo  "message" > file name.extension

viewing files;
       -use to see less pages:cat
       -use to display content in files: cat
       -use to see more pages:more
       -to see the last 4 lines: head 0n 4 file name.extension
